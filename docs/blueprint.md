# **App Name**: Hotel Suvidha Billing

## Core Features:

- Menu Item Management: Maintain a list of menu items with prices, allowing for easy selection during billing.
- Quantity Adjustment: Enable staff to increase or decrease the quantity of selected items.
- Automated Calculation: Automatically calculate the subtotal, total amount, and GST/tax based on selected items and quantities.
- Bill Generation: Generate a bill with a unique bill number, date, and the option to add the customer's name.
- PDF Export: Allow exporting the bill as a PDF document for printing or digital sharing.
- Responsive Design: Ensure the billing system is accessible and functional on mobile devices.

## Style Guidelines:

- Primary color: Soft Blue (#A0D2EB) to create a calm and professional environment.
- Background color: Light gray (#F0F0F0) to provide a clean and unobtrusive backdrop.
- Accent color: Teal (#008080) for interactive elements and call-to-action buttons.
- Body and headline font: 'PT Sans', a modern and readable sans-serif, for clarity.
- Use simple and intuitive icons for menu items and actions.
- A clean and organized layout to ensure easy navigation and usage.
- Subtle transitions and feedback animations for a smooth user experience.
